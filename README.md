## 2.2. SASS
- 실시간 감지하여 컴파일
  - `sass --watch [path]input.scss:[path]output.css [opt]`
- 컴파일 방법
  - `--style compressed` : 모두 한줄로 (minify)
  - **`--style compact`** :  공백 없이 속성별로 한 줄씩 
  - `--style expanded` : 일반적인 CSS 작성 방식; 스타일 별로 한 줄씩
